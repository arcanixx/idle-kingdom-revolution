# HERALDRY SYSTEM (original, pre-split)

## PURPOSE

Defines faction symbols and heraldic consistency rules.

Faction symbols must remain visually consistent across all assets.

Applies to:

- Heroes
- NPCs
- Enemies
- Portraits
- Buildings
- UI

---

# HERALDRY LOCK

A faction emblem is immutable.

The emblem never changes shape.

The emblem never changes pose.

The emblem never changes identity.

Only the following may evolve:

- material
- depth
- ornamentation
- lighting
- magical effects

---

# SHIELD SILHOUETTE LOCK

Shield Shape and Shield Surface are independent systems.

Shield Shape:
Defines physical shield silhouette.

Shield Surface:
Defines heraldry, decoration and effects.

Shape never changes because of rarity.

Surface may evolve because of rarity.

---

# STAFF SILHOUETTE LOCK

The staff silhouette is immutable.

The staff remains the same length.

The staff remains the same shape.

Only materials, decorations and magical effects evolve.

Rarity must never redesign the staff.

---

# EXAMPLE

Human Warrior

Common:
Blue Kite Shield

Mythic:
Blue Kite Shield

The shape remains identical.

Only materials and effects evolve.

---

# CLASS EQUIPMENT LOCK

Shield shape belongs to class design.

Not rarity.

Examples:

Warrior:
Kite Shield

Paladin:
Heater Shield

Guardian:
Tower Shield

Infantry:
Round Shield

Knight:
Kite Shield

Templar:
Heater Shield

Rarity never changes shield shape.

Class defines shield shape.

Rarity defines shield quality.
