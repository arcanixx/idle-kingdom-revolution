"Idle Kingdom Revolution - Idle Strategy Game"  
""  
"## Features"  
"- Automated battles with wave progression"  
"- Army formation management"  
"- Resource mining and collection"  
"- Castle defense mechanics"  
"- Player progression and leveling"  
"- Valor skill trees"  
"- Quests and achievements"  
"- Inventory and item system"  
"- Shop and marketplace"  
"- Mail notifications"  
"- Leaderboard rankings"  
""  
"## Tech Stack"  
"- Next.js 14 with TypeScript"  
"- Tailwind CSS for styling"  
"- Supabase for database"  
"- Zustand for state management"  
"- Radix UI components"  
""  
"## Development"  
"Install dependencies:"  
"```bash"  
"npm install"  
"```"  
""  
"Run development server:"  
"```bash"  
"npm run dev"  
"```"  
""  
"Build for production:"  
"```bash"  
"npm run build"  
"npm start"  
"```"
